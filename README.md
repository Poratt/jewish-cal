# Hebrew Calendar

Calendar with Jewish data - Shabbat times, holidays, daily times, and daily study schedules.

**Demo:** https://jewish-cal.netlify.app/

## Tech Stack
Angular 19 • TypeScript • Tailwind • PrimeNG • HebCal API

## Installation
```bash
npm install
ng serve
```

## Background
Learning project for Angular. Built it to practice working with external APIs and handling data & dates.
